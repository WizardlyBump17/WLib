### Build yourself

#### Pre-requisites

1. Git installed
2. Gradle installed
3. Java 8+ installed
4. [BuildTools](https://www.spigotmc.org/wiki/buildtools/) run for the versions: 1.8.8, 1.12.2, 1.13.2, 1.15.2, 1.16.5

#### Building

1. Clone the repository
2. Enter the directory
3. Run `gradle shadowJar`
